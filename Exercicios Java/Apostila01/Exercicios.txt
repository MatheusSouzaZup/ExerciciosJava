1. Qual a diferença entre JRE e JDK?
JDK, É o conjunto de ferramentas necessárias para realizar o desenvolvimento de aplicações java e inclui a JRE e ferramentas de programação. JRE, É composto pela JVM e pela biblioteca de classes Java utilizadas para execução de aplicações java, estas bibliotecas são chamadas de APIs Java.

2. Quais são os componentes da JDK?
javac – compilador
jar – empacotador
javadoc – ferramenta para geração de documentação
